Create a folder called "game_data" next to the program.

Place the following CSV files inside it:

Item.csv
BaseParam.csv
EquipSlotCategory.csv
ClassJobCategory.csv
Materia.csv
MateriaParam.csv
ItemLevel.csv

These files can be exported using SaintCoinach or Godbert.

Example folder structure:

ffxiv_solver/
    main.exe
    game_data/
        Item.csv
        BaseParam.csv
        EquipSlotCategory.csv
        ClassJobCategory.csv
        Materia.csv
        MateriaParam.csv
        ItemLevel.csv
